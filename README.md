# RedFlag - Fraud Detection Using SQL

# Project Overview
RedFlag is an industry-inspired SQL project focused on detecting fraudulent financial transactions using pure SQL. The project simulates the work of a fraud analyst at a fintech company by identifying suspicious transaction patterns from over 200,000 payment records.

This project demonstrates practical SQL skills including filtering, aggregation, grouping, conditional logic, subqueries, common table expressions (CTEs), and window functions.

# Objective
Develop a fraud detection engine capable of identifying multiple real-world fraud patterns without using Machine Learning or Python.

# Technologies Used
- MySQL 8.0
- MySQL Workbench
- SQL

# Results
Successfully detected multiple fraud patterns including:
- High transaction velocity
- Card testing attacks
- Refund abuse
- Money mule activity
- Structured transactions
- Geographic anomalies
- Velocity spikes
using SQL queries only.
